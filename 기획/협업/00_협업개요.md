# 협업(멀티 에이전트) 기록 — 개요

본 폴더는 메인 에이전트(Claude)가 외부 AI 모델과 **협업**한 내용을 정리한다.
질문 / 각 모델 답변 / 메인 에이전트의 판단·처리 내용을 기록한다.

## 협업 채널

| 호출 방법 | 모델 | 용도 |
|-----------|------|------|
| `agy -p "<프롬프트>"` (Antigravity CLI) | **Gemini** (3.1 Pro / 3.5 Flash) | 설계 리뷰·대안 제시 |
| `codex exec -m gpt-5.5 -s read-only --skip-git-repo-check "<프롬프트>"` | **GPT-5.5** (Codex) | 실제 코드 읽고 설계 리뷰 |

### 호출 시 주의(실측 노트)
- `codex exec` 는 **stdin 을 `< /dev/null` 로 막아야** 함(아니면 stdin 대기로 멈춤).
  또한 git 저장소가 아니면 **`--skip-git-repo-check`** 필요.
- `agy -p` 는 에이전트형이라 복잡한 질문에 **브라우저 MCP 도구(chrome-devtools-mcp)** 를
  띄우며 오래 멈출 수 있음 → 프롬프트에 "**Do NOT use any tools, answer in text**" 를
  명시하고 빠른 모델(Flash)을 쓰면 즉답.
- `--dangerously-skip-permissions`(agy) / `--dangerously-bypass-approvals`(codex) 류
  **안전장치 해제 플래그는 사용하지 않음**(권한 분류기 차단 + 불필요).
